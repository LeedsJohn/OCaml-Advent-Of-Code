# OCaml Advent of Code Solutions

If you want to run these solutions on your own input, copy and paste your input into `input/{two digit day number}.in`

Then, run `dune exec -- aoc [day number] [year] [optional section of problem to run]`

The options for which section of the problem to run are:

* test (Tests provided by Advent of Code)
* real (Your input file)
* all (Both tests and real input)

Append 1 or 2 to this argument if you only want to run it for one part - ex: `dune exec -- aoc 1 2023 all1`

I can't guarantee that all of these solutions will be good OCaml :)
